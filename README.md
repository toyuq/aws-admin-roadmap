# AWS-ADMIN-ROADMAP

## 프로젝트 목적
이 프로젝트는 기본적인 파이썬 애플리케이션을 구현하기 위한 것입니다. 이 애플리케이션은 특정 기능을 수행하며, 사용자가 쉽게 이해하고 사용할 수 있도록 설계되었습니다.

## 추가 문서

- `docs/aws-admin-roadmap.md` : AWS Admin 입문 로드맵
- `docs/aws-admin-journey/` : 단계별 학습 및 진행 기록 폴더

## 설치 방법
1. 이 저장소를 클론합니다:
   ```
   git clone <repository-url>
   ```
2. 프로젝트 디렉토리로 이동합니다:
   ```
   cd aws-admin-roadmap
   ```
3. 필요한 패키지를 설치합니다:
   ```
   pip install -r requirements.txt
   ```

## 사용법
애플리케이션을 실행하려면 다음 명령어를 사용하세요:
```
python src/main.py
```
