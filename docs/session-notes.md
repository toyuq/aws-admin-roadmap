# 세션 요약

## 오늘 이야기한 핵심
- AWS Admin은 단순히 AWS를 다루는 사람이 아니라, 안전하게 운영하고 문제를 재현/복구 가능하게 만드는 역할이다.
- 필요한 기본 지식: AWS 핵심 서비스, IAM/보안, 네트워크, 배포/운영, 비용 관리, IaC.
- 필요한 마음가짐: 최소 권한, 변경 기록, 되돌릴 수 있는 운영, 추측보다 확인, 재발 방지 중심.
- 프로젝트 표기는 `AWS-ADMIN-ROADMAP`으로 통일한다.

## 이어서 공부할 때 기억할 점
